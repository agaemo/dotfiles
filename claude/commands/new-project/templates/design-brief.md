## 4. ブランドアーキタイプ

| アーキタイプ | キーワード | 視覚的特徴の傾向 |
|------------|----------|----------------|
| **クリエイター** | 表現・独創性・芸術 | 個性的なタイポ、非対称レイアウト、視覚的な遊び |
| **ヒーロー** | 挑戦・勝利・力強さ | 高コントラスト、太いウェイト、ダイナミックな構図 |
| **セージ** | 知識・明確さ・知性 | 精緻なグリッド、落ち着いたトーン、情報密度の高い設計 |

| **ルーラー** | 権威・秩序・品格 | 対称性、上品なセリフ体、抑制されたカラー |
| **ケアギバー** | 温かさ・共感・サポート | ソフトなカラー、丸みのある形、親しみやすいタイポ |
| **マジシャン** | 変革・神秘・驚き | 予期しない色彩展開、劇的なスケール差、暗めのトーン |
| **ジェスター** | 楽しさ・遊び心・エネルギー | 鮮やかな色、大胆な形、予想を外す要素 |
| **エブリマン** | 誠実さ・親近感・実用性 | 飾らないシンプルさ、温かみのある中性色、直感的なパターン |
| **アウトロー** | 反骨・破壊・独立心 | 既存パターンを崩す構成、鋭いコントラスト、エッジな要素 |
| **ラバー** | 親密さ・美・感性 | 豊かな色彩、繊細なテクスチャ、感情的な曲線 |
| **イノセント** | 純粋さ・楽観・清潔感 | 明るいパステル、シンプルな形、余白を活かした軽さ |
