## 進め方

1. まず5〜6案を提示する
2. 各案について WebSearch で競合調査を行い、競合マトリクスを示す（全案並列で調査する）
3. 競合が強い案（❌ レッドオーシャン）は除外理由を記録し、別案に差し替える
4. 各案に採算試算（単価 × 潜在顧客数 × 想定シェア）を含める
5. 「別案がほしい」と言ったら、ヒアリングに戻らず案出し → 競合調査を繰り返す
6. 検討結果は最後に `docs/ideas.md` に保存する
