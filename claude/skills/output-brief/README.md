# output-brief パターンカタログ

## レイアウト

### simple-list
行ごとにアイテムを縦に並べるシンプルなリスト。情報量が多い場合や、テキスト中心のコンテンツに向く。

### panel
カード形式でアイテムをグリッド配置。視覚的なスキャンがしやすく、同程度の重要度の項目を並べるのに向く。

---

## カラーテーマ

| テーマ | 配色イメージ |
|--------|-------------|
| `warm`  | オレンジ・ベージュ。温かみがある |
| `cool`  | パープル・ラベンダー。落ち着いた印象 |
| `blue`  | 青系。清潔感・信頼感 |
| `green` | 緑系。自然・安心感 |
| `dark`  | 暗い背景。モダン・集中感 |
| `light` | 白ベース。シンプル・クリーン |
