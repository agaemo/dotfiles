    Claude Code ツール比較

    サンプルドキュメント — output-brief / table レイアウト

    ファイル操作ツール

    ツール主な用途Read 前提出力形式備考
    Readファイル読み込み—行番号付きテキストPDF・画像・Notebook 対応
    Writeファイル書き込み（新規・上書き）既存ファイルは必須—新規作成時は Read 不要
    Edit差分編集必須—replace_all でリネーム可
    Bashシェルコマンド実行—stdout/stderr副作用に注意

    検索・調査ツール

    ツール主な用途速度特徴
    Bash (grep)テキスト検索高速正規表現対応、ファイル名絞り込み可
    Bash (find)ファイル探索高速パターン・日時・サイズで絞り込み
    WebSearch外部情報検索中速最新情報・公式ドキュメント取得に有効
    WebFetchURL コンテンツ取得中速HTML → テキスト変換済みで返却

    エージェント・タスクツール

    ツール用途並列実行コスト
    Agentサブエージェント起動可（複数同時）高
    TaskCreateタスク作成・追跡—低
    Monitorファイル・プロセス監視可（persistent）低〜中
