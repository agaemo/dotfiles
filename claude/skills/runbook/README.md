# runbook

開発タスクの「読み上げ式チェックリスト」を生成するClaude Codeスキル。

## 使い方

```
/runbook 本番デプロイ
/runbook DBマイグレーション
/runbook インシデント対応
```

引数なしで起動するとタスク名を確認します。

## 出力

タスクのリスクレベル（HIGH / MEDIUM / LOW）を自動判定し、
以下のセクション構成でMarkdownチェックリストを生成します。

- **事前確認** — 実行前の確認事項
- **実行手順** — 順番通りに実行するステップ
- **完了確認** — 正常完了の確認
- **ロールバック手順** — 問題発生時の復旧手順（HIGH / MEDIUM のみ必須）

## 設計思想

航空SOP（Standard Operating Procedure）のチェックリスト文化を転用。
熟練者の暗黙知をチェックポイントの列に変換し、経験年数によらず同一品質の実行を保証する。
